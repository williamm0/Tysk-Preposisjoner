# Tyske Preposisjoner

En norsk nettside for å lære tyske preposisjoner og kasus. Siden er laget for elever i tysk på 10. trinn, med teori, øving, prøve, spill, flashkort, ordliste, statistikk og prestasjoner.

Nettsiden fokuserer på tre hovedgrupper:

- akkusativ-preposisjoner
- dativ-preposisjoner
- Wechselpräpositionen, altså preposisjoner som kan styre både akkusativ og dativ

## Hva siden inneholder

### Lær

Teorisiden forklarer de viktigste reglene for tyske preposisjoner. Den dekker akkusativ, dativ, Wechselpräpositionen, artikkelbøying, husketips og lesestoff.

Eksempler på temaer:

- `durch`, `für`, `gegen`, `ohne`, `um` styrer alltid akkusativ
- `aus`, `bei`, `mit`, `nach`, `seit`, `von`, `zu`, `außer`, `gegenüber`, `ab` styrer alltid dativ
- Wechselpräpositionen tar akkusativ ved bevegelse og dativ ved ro eller plassering
- `Wohin?` betyr bevegelse og gir akkusativ
- `Wo?` betyr sted eller ro og gir dativ
- bestemt artikkel bøyes ulikt i nominativ, akkusativ og dativ

### Øving

Øvingssiden har flere oppgavetyper og vanskelighetsgrader. Den er laget for å trene både forståelse og raske valg av riktig preposisjon, kasus og artikkel.

Oppgavene dekker blant annet:

- flervalg
- fyll inn riktig preposisjon
- velg riktig kasus
- artikkelbøying
- setningseksempler med forklaring
- akkusativ, dativ og Wechselpräpositionen blandet sammen

### Prøvemodus

Prøvemodus simulerer en ekte prøve. Den har tidtaker og automatisk retting, slik at du kan teste deg selv før en skoleprøve.

### Fremgang

Fremgangssiden viser statistikk og prøveresultater, slik at du kan se hva du har øvd på og hvor mye du har forbedret deg.

### Spill

Spillsenteret inneholder flere små spill for å gjøre øving mer variert. Siden har blant annet quiz, memory, fangstspill, skriveduell og andre spillbaserte oppgaver.

### Flashkort

Flashkort lar deg bla gjennom preposisjonene én om gangen. Du kan sortere ord du kan og ord du må øve mer på.

### Ordliste

Ordlisten gir en samlet oversikt over preposisjonene med norsk betydning, kasus og eksempelsetninger.

### Prestasjoner

Siden har et prestasjonssystem med 27 prestasjoner som kan låses opp ved å øve, spille og bruke nettsiden.

## Kasusgruppene

### Akkusativ

Disse preposisjonene styrer alltid akkusativ:

```text
durch · für · gegen · ohne · um
```

Eksempel:

```text
Das Geschenk ist für dich.
Wir gehen um den See.
```

I akkusativ er det bare maskulin bestemt artikkel som endrer seg:

```text
der → den
die → die
das → das
```

### Dativ

Disse preposisjonene styrer alltid dativ:

```text
aus · bei · mit · nach · seit · von · zu · außer · gegenüber · ab
```

Eksempel:

```text
Ich fahre mit dem Bus.
Er kommt aus Deutschland.
```

I dativ endrer alle artiklene seg:

```text
der → dem
die → der
das → dem
die → den
```

### Wechselpräpositionen

Disse preposisjonene kan styre både akkusativ og dativ:

```text
an · auf · hinter · in · neben · über · unter · vor · zwischen
```

Regelen er:

```text
Wohin? → akkusativ
Wo? → dativ
```

Eksempel:

```text
Ich gehe in die Schule.      // bevegelse, akkusativ
Ich bin in der Schule.       // ro/sted, dativ
```

## Artikkelbøying

| Kasus | Maskulin | Feminin | Nøytrum | Plural |
| --- | --- | --- | --- | --- |
| Nominativ | der | die | das | die |
| Akkusativ | den | die | das | die |
| Dativ | dem | der | dem | den |

## Hvordan kjøre nettsiden lokalt

Dette er en statisk nettside. Den trenger ikke installasjon eller bygging.

Last ned repoet og åpne `index.html` i nettleseren:

```bash
git clone https://github.com/williamm0/Tysk-Preposisjoner.git
cd Tysk-Preposisjoner
open index.html
```

På Windows kan du åpne `index.html` direkte fra filutforskeren.

## Filstruktur

```text
index.html          Forside
learn.html          Teori og forklaringer

progress.html       Statistikk og fremgang

ordliste.html       Ordliste
achievements.html   Prestasjoner
css/                Stilark
js/                 Data, logikk, lyd, innstillinger og prestasjoner
```

## Mål

Målet med nettsiden er å gjøre tyske preposisjoner lettere å lære. I stedet for bare å pugge lister får du forklaringer, eksempler, oppgaver, spill og fremgangssporing på samme sted.
